mapReduce (...)
Signature: decomposeF:('I -> Cloud<ContainerState<'I,'T>>) -> mapF:('T -> Cloud<'R>) -> reduceF:('R -> 'R -> Cloud<'R>) -> identity:(unit -> Cloud<'R>) -> input:'I -> Cloud<'R>
Type parameters: 'I, 'T, 'R

A distributed divide-and-conquer MapReduce workflow. Branches out until cluster size is exhausted; thereby continuing with thread local parallel semantics. Works with any input that can be decomposed to a binary tree-like structure.
